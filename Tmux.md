# tmux shortcuts & cheatsheet

new session

    tmux new -s name

attach session

    tmux a -t name
  
Window operations

    cnrl + b --> Command mode
    cmnd , % -->  vertical Split
    cmnd , " --> horizontal switch
    cmnt , arrow keys --> switch contronl inside session
    
    
Copy paste


    cmnd , [, then scroll to the required pos, then cntrl+space to select, cntrl+w to copy
    cmnt , ] to paste
